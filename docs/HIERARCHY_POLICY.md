# Иерархия стратегия → тактика (политика оператора)

Зафиксировано 11.06.2026 решением оператора. Менять только новым явным решением.

## Правила

1. **OracAI = стратегия (месяцы), planner = тактика (день-неделя).**
   Тактика никогда не торгует против стратегии в опасную сторону.

2. **Режим BULL → тактический SHORT невозможен.** Двойная защита:
   контрарианская ветка вершины требует regime != BULL, плюс
   `enforce_hierarchy()` как последний рубеж на всех путях вердикта.
   «Вершинность» при BULL = фиксация прибыли и стопы, не разворотная ставка.

3. **Режим BEAR → тактический LONG РАЗРЕШЁН** (асимметрия сознательная,
   подтверждена оператором 11.06: «иначе не поймать разворот»). Условия
   жёсткие: фаза дна (CAPITULATION/ACCUMULATION/LATE_BEAR) + oversold.
   Логика: развороты рождаются в медвежьем режиме; ждать официального BULL —
   гарантированно опаздывать на каждое дно. Согласовано с лестницей цикла,
   которая тоже покупает страх. Первое живое срабатывание: 11.06 02:53,
   LONG BTC @61,544 / LONG ETH @1,623 (ACCUMULATION, RSI 24-25).

## Измеримость

Оба правила — под замером, не на веру: A/B-журнал вердиктов (raw vs final)
копит статистику запрещённых шортов при BULL и разрешённых лонгов при BEAR.
Пересмотр политики — только по этим данным.
